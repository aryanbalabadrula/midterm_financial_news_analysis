# Midterm Project: Financial News & Stock Data
This repository contains my midterm submission for a financial data analysis project involving news data and stock market data.

## Files
midsem_submission.ipynb: Main notebook with news collection, merging with stock data,basic analysis completed through the 7 tasks.

news_raw.csv: Raw financial news collected from RSS feeds over a period of ten days.

news_cleaned.csv: Cleaned news data with formatted dates and headline length.

merged_midterm_data.csv: Final merged dataset with news and stock data, including trading day information.

## Tools used
Python, pandas, BeautifulSoup, requests, yfinance, matplotlib
